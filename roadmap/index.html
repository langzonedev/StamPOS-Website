Public roadmap

Built deliberately, released when it is ready.

The roadmap reflects the current product direction. Priorities may move as field testing, deployment planning and operator feedback reveal what matters most.

Complete

Core offline POS

Touch sales interface, local SQLite data, staff PIN access, product management, transactions, receipt printing and technician tools.

Complete

Linkly payment foundation

Cloud PIN pad pairing, token authentication, purchases, refunds, transaction status and conservative unknown-outcome handling.

Complete

1.0.0 controlled baseline

Version metadata, payment workflow, recovery handling, receipts, refunds, reporting and technician evidence tools aligned for controlled validation.

In progress

Field validation and cash-up refinements

Structured operator testing, reconciliation checks and event-day cash-up refinement before wider availability.

Planned

Public release pathway

Installer downloads, documentation, release notes, support processes and a customer-ready general-availability release.

Future

Broader deployment options

Additional hardware validation, multi-terminal considerations and optional services where they improve reliability without compromising offline operation.
