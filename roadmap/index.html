Public roadmap

Built deliberately, released when it is ready.

The roadmap reflects the current product direction. Priorities may move as field testing, accreditation and operator feedback reveal what matters most.

Complete

Core offline POS

Touch sales interface, local SQLite data, staff PIN access, product management, transactions, receipt printing and technician tools.

Complete

Linkly sandbox foundation

Cloud PIN pad pairing, token authentication, purchases, refunds, transaction status and conservative unknown-outcome handling.

In progress

Linkly accreditation

Startup recovery, logon, settlement, mandatory test execution and evidence collection for Cloud REST accreditation.

In progress

Event cash-up refinements

AUD note tendering, repeated note taps, clear change due and multi-day event cash-up without premature final closure.

Planned

Public release pathway

Installer downloads, documentation, release notes, support processes and an initial customer-ready StamPOS 1.0 release.

Future

Broader deployment options

Additional hardware validation, multi-terminal considerations and optional services where they improve reliability without compromising offline operation.
