Support matrix

Track tested StamPOS configurations as they are validated.

The support matrix records exact combinations of StamPOS version, Windows environment, terminal hardware, printer, payment provider and test status. It should be updated whenever a new physical or virtual setup is tested.

Recommended format

Keep the public support matrix in this version-controlled website so every change is visible in Git. A spreadsheet can still be used internally for detailed test evidence, screenshots and sign-off notes, but the public support position should remain a simple page that customers and technicians can read without downloading a file.

Minimum information for a new row

Record the StamPOS version, Windows edition, physical or virtual hardware, receipt printer, cash-drawer path if applicable, payment provider, test scope, result and any setup notes. Avoid marking a configuration as supported until the actual hardware path has been exercised.
