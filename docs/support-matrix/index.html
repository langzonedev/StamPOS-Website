Supported configurations

Hardware and software known to work with StamPOS.

Use this page to check the currently supported Windows, receipt-printer and payment setup before deployment.

Component Supported configuration Notes
StamPOS Version 1.0.0 Windows desktop release.
Operating system Windows 11 Pro Keep Windows and manufacturer drivers current.
Receipt printer Star TSP100-series, including TSP100IIIW Supported through the configured Windows or network-printing workflow. Confirm cutting and cash-drawer settings during installation.
Payments Linkly Cloud REST Requires a compatible Linkly-supported terminal, merchant onboarding and internet access.
Cash drawer Printer-connected drawer where supported by the selected printer Compatibility depends on the printer port, cable and drawer-pulse configuration.

Check your setup

Hardware models and merchant payment arrangements can vary. Contact StamPOS before purchase or installation when your computer, printer, cash drawer or payment terminal is not listed above.
