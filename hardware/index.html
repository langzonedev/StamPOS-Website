Hardware

Designed for practical Windows POS setups.

StamPOS is intended to work with standard commercial hardware while keeping setup and support understandable.

Core platform

Windows terminal

A Windows 11 PC, touchscreen terminal or laptop can run StamPOS. Final production requirements will be published with the first supported installer.

Validated path

Receipt printers

StamPOS supports Windows printing and direct network printing workflows. Star TSP100-series testing is part of the current deployment work.

Supported setup

Cash drawers

Cash-drawer support will follow the selected receipt-printer connection and command path, with deployment guidance published per supported model.

Integration

EFTPOS terminals

Linkly provides the first payment-terminal integration path. Compatible terminal availability depends on the merchant's acquirer and Linkly deployment.

Optional

Barcode scanners

USB scanners that present as keyboard input are a natural future extension where product workflows require barcode entry.

Future

Customer displays

Secondary displays and customer-facing sale information may be added where they improve the event and counter experience.

Compatibility approach

Support is based on tested combinations, not vague claims.

Before public release, StamPOS will publish a hardware matrix showing the Windows version, printer model, connection type, payment path and any required setup notes for each validated configuration.
