Hardware and support

Minimum specs and tested configurations.

StamPOS is intended for practical Windows POS workstations. Support is tracked by tested hardware, printer, payment and software-version combinations rather than broad compatibility claims.

Minimum specs

Current controlled 1.0.0 baseline.

These are starting requirements for validation and pilot planning. The supported list will tighten as more physical combinations are tested.

Operating system

Windows 11 Pro tested

Current validation has been performed on Windows 11 Pro in both virtual machine and physical hardware environments. 64-bit Windows 10/11 remains the packaging target until validation narrows the supported list.

Memory and storage

4 GB RAM minimum

Use 4 GB RAM and 2 GB free local disk as the minimum starting point. 8 GB RAM, SSD storage and at least 10 GB free space are recommended for event-day headroom, logs, backups and updates.

Display and input

Touch-friendly display

A touchscreen terminal or Windows PC with mouse input can be used. Plan for at least 1366 x 768 resolution; larger touch displays are recommended for fast product-tile operation.

Receipt printing

Star TSP100 path

Star TSP100-series receipt-printer testing is part of the current validated path. Printer driver, paper, cutter and cash-drawer pulse settings should be confirmed for each site.

Payments

Linkly first

Linkly is the first payment-terminal provider path. Card transactions require working internet access, merchant/acquirer onboarding and the correct Linkly production or sandbox credentials.

Peripherals

Record each device

Cash drawers, scanners, customer displays and other peripherals should be added to the support matrix only after the actual connection path and StamPOS version have been tested.
