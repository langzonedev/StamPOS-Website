Private beta

StamPOS is under controlled testing and is not yet available for general production use.

The product, hardware workflows and Linkly payment integration are still being validated. Public website demonstrations do not process real transactions. Installer access is restricted to approved testers under written beta terms.

Testing

Current status

StamPOS remains pre-release software. Features, data structures, hardware compatibility and user workflows may change before version 1.0.

Accreditation pending

Payments

Linkly integration testing and any required accreditation must be completed before general live EFTPOS use. Beta builds default to sandbox or simulator testing unless a live pilot is expressly approved.

No public installer

Controlled distribution

Executables are supplied only to named testers for approved devices and test periods. Redistribution and unauthorised production use are prohibited.

Public demo

Website demonstration

The interactive sale shown on this website is a browser demonstration only. It does not connect to a payment service, save a real sale or represent production approval.

Test safeguards

Tester responsibilities

Approved testers use test data where practical, maintain independent records and backups, report defects promptly and stop testing if a payment, security or data-integrity issue occurs.

Documentation

Beta testing documents
